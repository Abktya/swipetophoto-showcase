# SwipeToPhoto 📷

> Native iOS photo organisation app — live on the App Store

## Overview

A native iOS app for intuitive photo organisation using swipe gestures. Built entirely in Swift/SwiftUI as a solo project with AI-assisted development.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Language | Swift |
| UI Framework | SwiftUI |
| Photo Access | PHPhotoLibrary |
| Monetisation | AdMob (Google Mobile Ads SDK) |
| IAP | StoreKit (premium tier) |
| Dependencies | CocoaPods |
| AI Tools | Claude Code, Cursor |

## Key Features

- **Swipe-based browsing** — intuitive up/down gesture navigation
- **Photo editing** — filters, brightness, contrast, saturation
- **Duplicate finder** — detect and remove duplicate photos
- **Premium tier** — in-app purchase for advanced features
- **GDPR compliance** — Google UMP consent flow
- **Bilingual** — English and Turkish UI

## AI-Assisted Development

- **Claude Code** — SwiftUI component development, gesture handling, PHPhotoLibrary integration
- **Cursor** — Swift refactoring and optimisation

## Status

🟢 **Live on the Apple App Store**
